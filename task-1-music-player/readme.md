## 🎵 Task 1 - Music Player Web App

A simple **music streaming web application** built using:

- HTML  
- CSS (Tailwind CSS)  
- JavaScript  
- Audius API (for music streaming)

### ✨ Features:
- 🎧 Search songs online  
- ❤️ Save songs to library  
- 📚 View saved library  
- ▶️ Play music with player page  
- 📱 Responsive UI  

### 📁 Folder Structure:

│
├── index.html
